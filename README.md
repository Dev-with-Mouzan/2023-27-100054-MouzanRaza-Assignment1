# Assignment 1: Android Views Demonstration

## Project Overview
This is a comprehensive Android application demonstrating all major Android UI Views for academic submission. The project follows clean architecture principles, uses ViewBinding, and implements proper navigation between activities.

## Project Details
- **Name:** Assignment1_AndroidViews
- **Language:** Java
- **Minimum SDK:** 21 (Android 5.0 Lollipop)
- **Target SDK:** 34
- **Architecture:** Clean Architecture with ViewBinding

## Features Demonstrated

### Basic Views
1. **TextView** - Displaying text with various styles
2. **EditText** - User input fields with different input types
3. **Button** - Clickable buttons with event handling
4. **ImageView** - Image display with different scale types

### Input & Selection Views
5. **CheckBox** - Multiple selection checkboxes
6. **RadioButton** - Single selection radio groups
7. **Switch** - Toggle switches for settings
8. **ToggleButton** - On/off toggle buttons

### Adapter Views
9. **Spinner** - Dropdown selection list
10. **ListView** - Scrollable list of items
11. **RecyclerView** - Advanced list with ViewHolder pattern
12. **GridView** - Grid layout for items

### Progress & Feedback
13. **ProgressBar** - Indeterminate and determinate progress
14. **SeekBar** - Slider for value selection
15. **RatingBar** - Star rating widget

### Layout Demonstrations
16. **LinearLayout** - Vertical and horizontal orientation
17. **RelativeLayout** - Relative positioning
18. **ConstraintLayout** - Constraint-based positioning
19. **FrameLayout** - Layered layout
20. **ScrollView** - Scrollable content container

## How to Run

1. **Open in Android Studio:**
   - File → Open → Select the Assignment1_AndroidViews folder

2. **Sync Gradle:**
   - The project will automatically sync dependencies
   - Wait for the sync to complete

3. **Run the App:**
   - Connect an Android device or start an emulator
   - Click the Run button (Green triangle) or press Shift+F10
   - Select your device/emulator

## Usage

1. **Main Menu:**
   - The app opens with MainActivity showing a categorized list of all views
   - Buttons are organized into sections

2. **Navigation:**
   - Click any button to open the corresponding view demonstration
   - Each activity has a back button in the ActionBar

3. **Interactive Features:**
   - All demonstrations are fully functional
   - Try clicking, entering text, selecting items, and adjusting controls
